
# spider-flow-mailbox
#### 介绍
spider-flow mailbox插件


####  导出插件代码
[源码地址](https://gitee.com/jmxd/spider-flow-mailbox)

#### 使用方法
导入db文件夹sp_sp_mail.sql文件到数据库

在 spider-flow-web/pom.xml引用  
```java
<dependency>
 	<groupId>org.spiderflow</groupId>
 	<artifactId>spider-flow-mailbox</artifactId>
</dependency>
```
